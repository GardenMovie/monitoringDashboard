# Monitoring Dashboard

This project is a Next.js-based monitoring dashboard for visualizing server or application metrics (CPU, RAM, Disk, Ping) over time. It features interactive charts, a modern UI, and real-time data updates via API endpoints.

- Built with Next.js and React
- Uses Tailwind CSS for styling
- Data is fetched from RESTful API endpoints under `/api/charts/`
- Supports daily, hourly, and minutely metric views
